Known issues (closed beta)

Unlisted page for beta testers. Update this list as issues are fixed. Last updated: July 16, 2026

This page is not linked from the public home, How to, Privacy, or Terms pages. Prefer reporting new bugs to asteroid@jlundmark.org.

Current notes

  • English only in the first closed beta UI.
  • Google Drive as audiobook library is disabled in this release (Dropbox / OneDrive / on-device). Optional Google sign-in is for progress/bookmark sync (Drive app data).
  • Large first-time libraries can take a long time to index/scan/resolve. Keep the app usable (foreground or playing) during the first Index pass when possible.
  • Metadata matching depends on folder names and tags; some books need a manual Author/Title edit + resolve.
  • Rough edges expected — please report crashes with version/build from Settings → About.

Recently addressed (may still need a Play update)

  • Latest Played: finished books stay on the list; list refresh is less jumpy during playback.
  • Advertising ID declaration aligned with Firebase Analytics (see Privacy Policy).

How to report

Email asteroid@jlundmark.org with: what you did, what you expected, what happened, device model, Android version, and app version/build.
